Bem-vindo ao Painel de Navegação

Selecione a ferramenta desejada na barra de navegação acima para gerenciar suas conversas.

Como Conectar uma Instância

1
Selecione uma Ferramenta
Clique em uma das ferramentas na barra de navegação (Typebot, OpenAI, Dify ou EvolutionBot) para acessar o painel correspondente.
2
Gerenciar Instâncias
No painel da ferramenta, clique no botão "Gerenciar Instâncias" localizado no canto superior direito da tela.
3
Adicionar Nova Instância
Preencha o formulário com as seguintes informações:
  • Nome do Cliente: Identificação da instância
  • Base URL: Endereço da API (ex: https://sua-api.com)
  • Nome da Instância: ID da instância no sistema
  • API Key: Chave de autenticação fornecida
4
Salvar e Selecionar
Clique em "Adicionar Instância" para salvar. Depois, selecione a instância adicionada no menu suspenso ao lado do botão "Gerenciar Instâncias".
Nota: Todas as instâncias criadas são salvas localmente no seu navegador. Se você limpar os dados do navegador ou acessar de outro dispositivo, precisará configurar as instâncias novamente.
